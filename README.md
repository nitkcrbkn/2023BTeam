# ROBOCON2023-B
高専ロボコン2023 Bチーム「TAKE採物語」の制御です

ロボ研の2rinテンプレを使用しています。


# 仕様

コントローラーはDUALSHOCK3を使用

## 使用MD

| MD番号 | つなぐモーター |
|------------|---------|
| 0   | 足まわり右   |
| 1   | 足まわり左   |
| 2       | 展開右     |
| 3       | 展開左     |

## 操作
| コントローラーの操作 | ロボットの動作 |
|------------|---------|
| 右スティック上下   | 足まわり右   |
| 左スティック上下   | 足まわり左   |
| L1         | 前輪上昇    |
| R1         | 前輪下降    |
