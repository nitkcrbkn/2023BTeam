# ROBOCON2023-B
高専ロボコン2023 Bチーム「TAKE採物語」の制御です

ロボ研の2rinテンプレを使用しています。

## 使用MD
| MD番号 | つなぐモーター |
|------|---------|
| 0    | メイン右タイヤ |
| 1    | 前右タイヤ   |
| 2    | メイン左タイヤ |
| 3    | 前左タイヤ   |
| 4    | パンタ上下   |
| 5    | お助け角度調整 |
| 6   | フルーツキャッチ |

## 使用GPIOピン
| GPIOピン | つなぐもの    |
|--------|----------|
| 13     | リミットスイッチ |
